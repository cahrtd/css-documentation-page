Introduction

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of HTML documents. It allows developers to control layout, colors, fonts, and other visual aspects of a webpage, separating content from design.

By linking a CSS file to your HTML, you can style multiple pages consistently and make your site responsive across different devices. CSS uses selectors to target HTML elements, applying styles that cascade based on specificity and inheritance rules.

Selectors

Selectors are patterns used to select the elements you want to style. There are various types of selectors, including simple selectors like element selectors (p, h1), class selectors (.class-name), and ID selectors (#id-name).

  • Element selector: targets elements by tag name 
    p {             
  color: blue;  
}
  • Class selector: targets elements with a specific class 
    .highlight {
  background-color: yellow;
}
  • ID selector: targets a unique element by ID 
    #header {
  font-size: 24px;
}

More advanced selectors include attribute selectors (input[type="text"]), pseudo-classes (:hover, :nth-child(n)), and pseudo-elements (::before, ::after). Understanding selectors helps you apply styles precisely without unnecessarily cluttering your CSS.

  • Attribute selector: targets elements with a specific attribute 
    input[type="text"] {
  border: 1px solid gray;
}
Box Model

The box model is fundamental to layout in CSS. Every HTML element is considered a rectangular box composed of content, padding, border, and margin areas.

  • Content is the actual content like text or images
  • Padding is the space inside the box between the content and border
  • Border surrounds the padding and content
  • Margin is the space outside the border that separates elements
div {
  width: 200px;
  padding: 20px;
  border: 5px solid black;
  margin: 10px;
}

Properly manipulating these areas is essential for spacing and alignment on your webpage.

Flexbox

Flexbox is a powerful layout module designed to arrange items in one dimension, either as a row or column. It simplifies complex layouts and provides flexibility for responsive designs.

Key properties include:

  • display: flex turns on flexbox on a container
  • justify-content aligns items horizontally (start, center, space-between)
  • align-items aligns items vertically within the container
  • flex-grow, flex-shrink, and flex-basis control how flex items resize and distribute space

Flexbox is widely supported and often preferred over older layout methods like floats.

Responsive Design and Media Queries

Responsive design adjusts layout for different screen sizes.

  • Use relative units like %, vw (viewport width), and vh (viewport height)
  • Use media queries to apply CSS rules based on device width
.container {
  width: 80%;
  margin: auto;
}
@media (max-width: 600px) {
  .container {
    width: 100%;
  }
}

This means on screens smaller than 600px, .container will take the full width.

CSS Animations

CSS animations enable you to create smooth and dynamic visual effects without needing JavaScript. You define a sequence of style changes with @keyframes, then apply the animation to elements using properties like animation-name, animation-duration, and animation-timing-function.

Animations can fade elements in or out, move them, rotate, scale, or combine multiple effects.

@keyframes slideIn {
  from {
    transform: translateX(-100%);
    opacity: 0;
  }
  to {
    transform: translateX(0);
    opacity: 1;
  }
}
.element {
  animation-name: slideIn;
  animation-duration: 1.5s;
  animation-timing-function: ease-out;
  animation-fill-mode: forwards;
}

Animations enhance user experience by adding interactivity and visual feedback, making websites feel more alive and engaging.
